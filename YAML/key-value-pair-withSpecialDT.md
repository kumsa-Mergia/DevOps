# Key-Value Pairs with Special Data Types

YAML supports various data types for values, such as strings, integers, floats, booleans, and more.

    - Strings: "Hello" or 'World'
    - Integers: 42
    - Floats: 3.14
    - Booleans: true or false
    - Null: null

## Example:

```yml
boolean: true
integer: 42
float: 3.14
null_value: null
```
