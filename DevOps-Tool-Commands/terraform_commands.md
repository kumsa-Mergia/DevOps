
# Terraform Commands

## 1. **Initialize a Terraform Working Directory**
This command initializes a working directory containing Terraform configuration files.
```bash
terraform init
```

## 2. **Validate Configuration Files**
This command checks whether the configuration files are valid.
```bash
terraform validate
```

## 3. **Create an Execution Plan**
This command generates an execution plan, showing what actions Terraform will take.
```bash
terraform plan
```

## 4. **Apply Changes**
This command applies the changes required to reach the desired state of the configuration.
```bash
terraform apply
```

## 5. **Destroy Terraform-managed Infrastructure**
This command destroys the infrastructure managed by Terraform.
```bash
terraform destroy
```

## 6. **Show the Current State**
This command displays the current state of your infrastructure.
```bash
terraform show
```

## 7. **List the Resources in the Current State**
This command lists all the resources in the current Terraform state.
```bash
terraform state list
```

## 8. **Get Provider Plugins**
This command downloads the required provider plugins.
```bash
terraform get
```

## 9. **Output Values from Terraform State**
This command shows output values from the current state.
```bash
terraform output
```

## 10. **Inspect the Current State**
This command inspects the current state of a resource or module.
```bash
terraform state show <resource_name>
```

## 11. **Import Existing Resources into Terraform**
This command imports an existing resource into Terraform management.
```bash
terraform import <resource_type>.<resource_name> <resource_id>
```

## 12. **Generate an Execution Plan (with Detailed Output)**
This command generates an execution plan with detailed output to help you understand each change Terraform is going to make.
```bash
terraform plan -out=plan.tfplan
```

## 13. **Execute a Plan**
This command applies a specific plan file generated by `terraform plan -out=plan.tfplan`.
```bash
terraform apply plan.tfplan
```

## 14. **Refresh the State**
This command updates the state with the latest changes from the real infrastructure.
```bash
terraform refresh
```

## 15. **Check the Provider Version**
This command checks the version of the provider used in the configuration.
```bash
terraform providers
```

## 16. **Upgrade Terraform Providers**
This command upgrades the providers to the latest versions that are compatible with your configuration.
```bash
terraform providers lock
```

## 17. **Generate and Display a Graph**
This command generates and displays a graph of the resources defined in your configuration.
```bash
terraform graph
```

## 18. **List the Modules Used**
This command lists the modules used by your Terraform configuration.
```bash
terraform modules
```

## 19. **Show Terraform Version**
This command displays the version of Terraform that is installed.
```bash
terraform version
```
